
Esta aplicación permite explorar información de países en tiempo real, utilizando NestJS para el Backend y React para el Frontend.

Funcionalidad
Listado de países: Datos obtenidos desde la API externa.
Búsqueda en tiempo real: Filtrado por nombre mientras el usuario escribe.
Filtro por región: Clasificación de países por continente.
Ordenamiento dinámico: Por nombre (A-Z, Z-A) y por cantidad de población.
Diseño Responsivo: Interfaz construida con Bootstrap.

Tecnologías Utilizadas
Backend: NestJS, Axios, RxJS.
Frontend: React (Vite), React-Bootstrap, Axios.
